## Project

AMC Sales Estimating System — a web app for estimating sales figures/quotes.

## Tech stack

- Next.js 15, TypeScript
- Tailwind CSS, shadcn/ui, Lucide Icons
- Supabase (PostgreSQL) with Drizzle ORM
- React Hook Form + Zod for forms/validation
- TanStack Table for data tables
- Zustand for client state
- Deployed on Vercel

## Conventions

- Package manager: pnpm (not npm or yarn)
- Commit messages follow Conventional Commits (`feat:`, `fix:`, `chore:`, etc.)

This project is early-stage; update this file as architecture, commands, and conventions solidify.
